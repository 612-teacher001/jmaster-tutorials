表示確認用リンク

このページはアプリケーションを開発していく過程で表示確認を行う際に必要となるURLの入力を低減するためのリンク集です。
チュートリアル計画表から作成しているので、実際には初期表示用のURLだけで十分です。

T0.00：プロジェクト構築

この機能は省略

T1.00：基本機能

T2.00：共通機能

T2.10 全件検索 アプリケーション初期表示 http://localhost:8080/tutorials-root/ProductServlet
商品を全件検索結果を表示 http://localhost:8080/tutorials-root/ProductServlet/list
T2.20 カテゴリ検索 DVD（categoryId=2）を選択した検索結果を表示 http://localhost:8080/tutorials-root/ProductServlet/list?categoryId=2
T2.30 並べ替え 価格の安い順（sortOrder=asc）に並べ替えた検索結果を表示 http://localhost:8080/tutorials-root/ProductServlet/list?sortOrder=asc
価格の高い順（sortOrder=desc）に並べ替えた検索結果を表示 http://localhost:8080/tutorials-root/ProductServlet/list?sortOrder=desc
T2.40 キーワード検索 キーワード「a」（keyword=a）による検索結果を表示 http://localhost:8080/tutorials-root/ProductServlet/list?keyword=a
T2.50 範囲検索 1000円以下の商品（maxPrice=1000）の検索結果を表示 http://localhost:8080/tutorials-root/ProductServlet/list?maxPrice=1000
