表示確認用リンク

このページはアプリケーションを開発していく過程で表示確認を行う際に必要となるURLの入力を低減するためのリンク集です。
チュートリアル計画表から作成しているので、実際には初期表示用のURLだけで十分です。

T0.00：プロジェクト構築
T1.00：基本機能
T1.10 JSPでアプリケーションタイトルを表示する http://localhost:8080/tutorials-root/BasicServlet
T1.20 入力部品を配置したページを表示する http://localhost:8080/tutorials-root/BasicServlet/input
T1.30 入力画面のボタンクリックによる画面遷移 http://localhost:8080/tutorials-root/BasicServlet/result
T2.00：共通機能
T2.10 商品を全件検索結果を表示 http://localhost:8080/tutorials-root/ProductServlet/list
T2.20 選択入力値による商品検索結果を表示 http://localhost:8080/tutorials-root/ProductServlet/list/?categoryId=
T2.30 並べ替え指定による検索結果を表示 http://localhost:8080/tutorials-root/ProductServlet/list/?keyword=
T2.40 キーワード入力による検索結果を表示 http://localhost:8080/tutorials-root/ProductServlet/list/?keyword=&maxPrice=
T2.50 範囲指定による検索結果を表示 http://localhost:8080/tutorials-root/ProductServlet/list/?keyword=&maxPrice=&sort=
T3.00：商品管理
T3.11 商品登録画面を表示する http://localhost:8080/tutorials-root/ProductServlet/insert/?categoryId=&name=&price=&quantity=
T3.12 登録内容確認画面を表示する http://localhost:8080/tutorials-root/ProductServlet/insert/?categoryId=&name=&price=&quantity=
T3.13 商品を登録する http://localhost:8080/tutorials-root/ProductServlet/update
T3.21 商品情報の変更画面を表示する http://localhost:8080/tutorials-root/ProductServlet/update/?categoryId=&name=&price=&quantity=
T3.22 変更内容確認画面を表示する http://localhost:8080/tutorials-root/ProductServlet/update/?categoryId=&name=&price=&quantity=
T3.23 商品情報を更新する http://localhost:8080/tutorials-root/ProductServlet/delete/?id=
T3.31 削除する商品確認画面を表示する http://localhost:8080/tutorials-root/ProductServlet/insert
T3.32 商品を削除する http://localhost:8080/tutorials-root/ProductServlet/delete/?id=
T4.00：カテゴリ管理
T4.11 カテゴリ登録画面を表示する http://localhost:8080/tutorials-root/CategoryServlet/insert
T4.12 カテゴリ内容確認画面を表示する http://localhost:8080/tutorials-root/CategoryServlet/insert/?name=
T4.13 カテゴリ登録する http://localhost:8080/tutorials-root/CategoryServlet/insert/?name=
T4.21 カテゴリ変更画面を表示する http://localhost:8080/tutorials-root/CategoryServlet/update
T4.22 変更内容確認画面を表示する http://localhost:8080/tutorials-root/CategoryServlet/update/?id=
T4.23 カテゴリを更新する http://localhost:8080/tutorials-root/CategoryServlet/update/?id=&name=
T4.31 削除するカテゴリ確認画面を表示する http://localhost:8080/tutorials-root/CategoryServlet/delete/?id=
T4.32 カテゴリを削除する http://localhost:8080/tutorials-root/CategoryServlet/delete/?id=
